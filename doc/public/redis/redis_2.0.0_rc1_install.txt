1. 如何在ubuntu10.04上安装redis?
<1> 到redis主页上下载redis - redis-2.0.0-rc1.tar.gz
    http://code.google.com/p/redis/

<2> 编译redis
    tar -xzvf redis-2.0.0-rc1.tar.gz
    cd redis-2.0.0-rc1
    make

<3> 安装redis
    redis目前没有安装脚本, 我们只需要把文件手动复制到目录即可:
    sudo mkdir /usr/local/redis
    sudo mkdir /usr/local/redis/bin
    sudo mkdir /usr/local/redis/etc
    sudo cp redis-server redis-cli redis-check-dump redis-check-aof
            redis-benchmark /usr/local/redis/bin
    sudo cp redis.conf /usr/local/redis/etc

<4> 启动redis server
    cd /usr/local/redis
    sudo ./bin/redis-server etc/redis.conf

    注意:
    默认复制过去的redis.conf文件的daemonize参数为no, 所以redis不会在后台运行. 
    修改为yes则为后台运行redis.
    另外配置文件中规定了pid文件, log文件和数据文件的地址, 如果有需要先修改, 默认log信息定向到stdout.

<5> 扩展配置:
    sudo mkdir /usr/local/redis/var
    sudo mkdir /usr/local/redis/var/log
    sudo mkdir /usr/local/redis/var/pid
    sudo mkdir /usr/local/redis/var/db
    修改: etc/redis.conf
    daemonize yes
    pidfile /usr/local/redis/var/pid/redis.pid
    logfile /usr/local/redis/var/log/redis.log
    dir /usr/local/redis/var/db

    sudo ./bin/redis-server etc/redis.conf 会在后台运行redis
    测试:
    bin/redis-cli set key1 val1
    OK
    bin/redis-cli get key1 
    "val1"
    bin/redis-cli get key2
    (nil)
    注意:
    在设置键对应的值的时候, 按照协议的规定是要提供数据大小这个参数的, 上面的redis-cli命令之所以没有提供这个参数
    是因为redis-cli本身进行了封装.

    使用telnet测试:
    telnet localhost 6379
    Trying ::1...
    Trying 127.0.0.1...
    Connected to localhost.
    Escape character is '^]'.
    set key2 4     
    val2
    +OK
    get key2
    $4
    val2

<6> 如何停止服务器?
    bin/redis-cli shutdown
