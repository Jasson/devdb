
OpenSSL


1. 关于key, 以及如何创建一个private key?
<1> key一般分为public key和private key, 在openssl中, private key中包含了
    public key的信息, 所以public key不需要单独创建.

<2> 如何创建一个private key?
    openssl genrsa -out privatekey.pem 2048        - 不需要密码保护
    
    openssl genrsa -des3 -out privatekey.pem 2048  - 需要密码保护
    Enter pass phrase for privatekey.pem:xxxxxx

2. 关于证书文件(certificates), 以及如何创建一个自签名的证书?
<1> 创建一个证书的流程是：
    > 创建一个private key.
    > 创建一个certificate signing request(证书请求), 这个需要private key,
      因为证书中需要包含public key, 创建的priavate key中有这些信息.
      openssl req -new -key privatekey.pem -out cacert.csr
    > 把创建好的证书请求拿到CA(certificate authority)证书认证机构审批.

<2> 创建一个自签名的证书：
    > 创建一个private key － privatekey.pem
    > openssl req -new -x509 -key privatekey.pem -out cacert.pem -days 1095 

3. 如何合并一个证书和一个privatekey文件?
<1> 直接赋值就可以, 证书和privatekey文件都是文本内容.
    cat < privatekey.pem >> cacert.pem
